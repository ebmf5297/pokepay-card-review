## **避坑指南：KYC、风控、退款那些事**

用 Pokepay 有几个坑得提前知道，免得踩了怪平台。

**1. KYC 别糊弄。** 信息和证件必须完全一致，姓名用护照就填英文，用身份证就填中文。Chrome 浏览器 + 开相机权限，通过率最高。虚拟卡基础 KYC 用身份证就行，实体卡建议直接上护照。

**2. 随用随充，别囤大额。** 这是所有加密卡的通用原则——账户里只留近期要用的额度，剩下的留在自己的钱包里。降低单一平台风险。

**3. 别在受限地区用。** 美国、俄罗斯、伊拉克这些地区 Pokepay 不服务，IP 也尽量别乱跳，订阅服务时 IP 最好对应卡区（港卡配港/亚太节点），否则容易触发风控。

**4. CVV 别多次输错。** 多次输错 CVV 或有效期，卡片会被锁定，要联系客服验证本人才能解锁。实体卡 PIN 码同理。

**5. 退款要有耐心。** Visa 退款走网络处理，普通商家 3–7 个工作日，跨境交易 7–30 天，争议退款（Chargeback）可能要 30–90 天。别以为退款没到就是平台黑你钱，先等 Visa 网络走完流程。

**6. 别碰 Steam、暴雪、EA。** 平台明确不支持，硬上大概率失败还可能触发风控。
